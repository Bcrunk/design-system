# @uq-uidf/core

> Core package that all other @uq-uidf packages depend on.

## Usage

### Installation

With Yarn:  
```shell
yarn add @uq-uidf/core
```

With NPM:  
```shell
npm i @uq-uidf/core
```

### Stylesheets

```scss
// SCSS

@import '~@uq-uidf/core/src/scss/_component';
```

```css
/* CSS (PostCSS) */
/* Watch this space... */
```

### Javascript

This package has no Javascript component.
