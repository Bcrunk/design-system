# `@uq-uidf/main-menu`

> Main menu component

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/main-menu
```

With NPM:
```shell
npm i @uq-uidf/main-menu
```

---

## Dependencies

```
@uq-uidf/main-menu
├─ @uq-uidf/core
├─ @uq-uidf/foundation-global
│  └─ @uq-uidf/core
└─ @uq-uidf/foundation-grid
   └─ @uq-uidf/foundation-global
      └─ @uq-uidf/core
```
