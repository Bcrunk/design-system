# `@uqmchu/main-menu`

> Main menu component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/main-menu
```

With NPM:
```shell
npm i @uqmchu/main-menu
```

---

## Dependencies

```
@uqmchu/main-menu
├─ @uqmchu/core
├─ @uqmchu/foundation-global
│  └─ @uqmchu/core
└─ @uqmchu/foundation-grid
   └─ @uqmchu/foundation-global
      └─ @uqmchu/core
```
