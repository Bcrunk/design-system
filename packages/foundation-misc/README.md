# `@uqds/foundation-misc`

> Foundation (5) misc. parts still in use by some other UIDF packages

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/foundation-misc
```

With NPM:
```shell
npm i @uqds/foundation-misc
```

---

## Dependency graph

```shell
@uqds/foundation-misc
└─ @uqds/foundation-global
   └─ @uqds/core
```
