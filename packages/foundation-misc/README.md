# `@uq-uidf/foundation-misc`

> Foundation (5) misc. parts still in use by some other UIDF packages

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/foundation-misc
```

With NPM:
```shell
npm i @uq-uidf/foundation-misc
```

---

## Dependency graph

```shell
@uq-uidf/foundation-misc
└─ @uq-uidf/foundation-global
   └─ @uq-uidf/core
```
