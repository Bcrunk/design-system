# `@uqmchu/foundation-misc`

> Foundation (5) misc. parts still in use by some other UIDF packages

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/foundation-misc
```

With NPM:
```shell
npm i @uqmchu/foundation-misc
```

---

## Dependency graph

```shell
@uqmchu/foundation-misc
└─ @uqmchu/foundation-global
   └─ @uqmchu/core
```
