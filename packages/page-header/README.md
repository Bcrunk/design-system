# `@uqmchu/page-header`

> Page Header component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/page-header
```

With NPM:
```shell
npm i @uqmchu/page-header
```

---

## Dependency graph

```shell
@uqmchu/page-header
├─ @uqmchu/core
└─ @uqmchu/foundation-global
   └─ @uqmchu/core
```
