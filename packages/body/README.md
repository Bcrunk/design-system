# @uq-uidf/body

> Basic styles for HTML elements (excluding form elements).

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/body
```

With NPM:
```shell
npm i @uq-uidf/body
```

### Stylesheets

```scss
// SCSS

@import '~@uq-uidf/body/src/scss/_component';
```

```css
/* CSS (PostCSS) */
/* Watch this space... */
```

### Javascript

This package has no Javascript component.
