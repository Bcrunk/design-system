# @uqmchu/body

> Basic styles for HTML elements (excluding form elements).

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/body
```

With NPM:
```shell
npm i @uqmchu/body
```

---

## Dependencies

```
@uqmchu/body
├─ @uqmchu/core
└─ @uqmchu/foundation-global
   └─ @uqmchu/core
```
