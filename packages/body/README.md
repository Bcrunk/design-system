# @uq-uidf/body

> Basic styles for HTML elements (excluding form elements).

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/body
```

With NPM:
```shell
npm i @uq-uidf/body
```

---

## Dependencies

```
@uq-uidf/body
├─ @uq-uidf/core
└─ @uq-uidf/foundation-global
   └─ @uq-uidf/core
```
