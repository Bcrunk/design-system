# `@uq-uidf/foundation-grid`

> Foundation (5) grid

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/foundation-grid
```

With NPM:
```shell
npm i @uq-uidf/foundation-grid
```
