# `@uqds/foundation-block-grid`

> Foundation (5) block grid

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/foundation-block-grid
```

With NPM:
```shell
npm i @uqds/foundation-block-grid
```

---

## Dependency graph

```shell
@uqds/foundation-block-grid
└─ @uqds/foundation-global
   └─ @uqds/core
```
