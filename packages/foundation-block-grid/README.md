# `@uq-uidf/foundation-block-grid`

> Foundation (5) block grid

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/foundation-block-grid
```

With NPM:
```shell
npm i @uq-uidf/foundation-block-grid
```
