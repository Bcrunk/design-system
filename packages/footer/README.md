# `@uqds/footer`

> Footer component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/footer
```

With NPM:
```shell
npm i @uqds/footer
```

---

## Dependency graph

```shell
@uqds/footer
├─ @uqds/core
├─ @uqds/foundation-global
│  └─ @uqds/core
└─ @uqds/foundation-misc
   └─ @uqds/foundation-global
```
