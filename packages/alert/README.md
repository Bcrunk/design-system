# `@uqds/alert`

> Alert component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/alert
```

With NPM:
```shell
npm i @uqds/alert
```

---

## Dependency graph

```shell
@uqds/alert-classic

├─ @uqds/foundation-global
│  └─ @uqds/core
└─ @uqds/foundation-grid
```
