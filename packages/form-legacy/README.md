# @uqds/form-legacy

> Deprecated styles for HTML form elements and form validation.

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/form-legacy
```

With NPM:
```shell
npm i @uqds/form-legacy
```

---

## Dependencies

```shell
@uqds/form-legacy
├─ @uqds/core
└─ @uqds/grid
   └─ @uqds/core
```
