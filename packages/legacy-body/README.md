# @uqds/legacy-body

> Basic styles for HTML elements (excluding form elements).

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/legacy-body
```

With NPM:
```shell
npm i @uqds/legacy-body
```

---

## Dependencies

```
@uqds/legacy-body
├─ @uqds/core
└─ @uqds/foundation-global
   └─ @uqds/core
```
