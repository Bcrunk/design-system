# @uqmchu/body

> Styles for HTML form elements and form validation.

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/form
```

With NPM:
```shell
npm i @uqmchu/form
```

---

## Dependencies

```shell
@uqmchu/form
├─ @uqmchu/core
└─ @uqmchu/foundation-global
   └─ @uqmchu/core
```
