# @uq-uidf/body

> Styles for HTML form elements and form validation.

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/form
```

With NPM:
```shell
npm i @uq-uidf/form
```

### Stylesheets

```scss
// SCSS

@import '~@uq-uidf/form/src/scss/_component';
```

```css
/* CSS (PostCSS) */
/* Watch this space... */
```

### Javascript

This package has no Javascript component.
