# @uq-uidf/card

> Card components

## Usage

### Installation

With Yarn:
```shell
yarn add @uq-uidf/card
```

With NPM:
```shell
npm i @uq-uidf/card
```

### Stylesheets

```scss
// SCSS

@import '~@uq-uidf/core/src/scss/_component';
@import '~@uq-uidf/button/src/scss/_component';
@import '~@uq-uidf/card/src/scss/_component';
```

```css
/* CSS (PostCSS) */
/* Watch this space... */
```

### Javascript

This package has no Javascript component.
