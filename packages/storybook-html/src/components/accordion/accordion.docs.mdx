## Overview

Accordion component helps users to see only the content they need. Accordions expand and collapse sections of content.

## How and when to use

- Use the default accordion example if you have preference for `div` elements in the components html code.
- Use default accordion which uses list tags if you have preference using `ul` and `li` elements in the components html code.
- Use the shaded accordion if you need the accordion toggle panels to be shaded light grey instead of white.
- Use secondary accordion example if you would like the accordion title font size to be smaller than the default.
