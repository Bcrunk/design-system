# Grid

## 6-8-12 grid

We've built a grid that adapts the number of available columns to different
screen sizes. It's a 6-, 8-, and 12-column grid built into one CSS class system!
Starting with mobile, the grid column classes, 

> By default, we add the [specified margins](?path=/docs/layout-layout--base-layout-grid-visualised#specifications-per-breakpoint)
> to both sides of the column row *and* apply a max-width on XL screens on the
> root `.uq-grid` node.

<Story height={80} id="layout-grid--twelve-eight-six" />

```html
<div class="uq-grid">
  <div class="uq-grid__col uq-grid__col--4 uq-grid__col--md-5 uq-grid__col--xl-8">Content</div>
  <div class="uq-grid__col uq-grid__col--2 uq-grid__col--md-3 uq-grid__col--xl-4">Content</div>
</div>
```

### Nesting

Nested `.uq-grid` elements don't have any margins or max. width applied.

<Story height={140} id="layout-grid--twelve-eight-six-nested" />

### Full-width 6-8-12 grid

If you want to manage the margins yourself i.e. nest the grid in your own custom
container, you can apply the `uq-grid--full-width` class as well.

#### For example

```html
<div class="uq-grid uq-grid--full-width">
  <div class="uq-grid__col uq-grid__col--7">
    <div>Your content</div>
  </div>
  <div class="uq-grid__col uq-grid__col--5">
    <div>Your content</div>
  </div>
</div>
```
