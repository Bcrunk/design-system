# Page header

The Page header component is used to display the page title or the breadcrumbs or a combination of both. 
The default background color is set to light grey to distinguish it from the page contents.

<Canvas withSource="none">
  <Story id="components-page-header--page-header" />
</Canvas>

### Page header default

```html
<div class="page__header">  
  <div class="row">
    <div class="large-12 columns">
      <h1 class="page__title">Page title</h1>
    </div>
  </div>
</div>
```
