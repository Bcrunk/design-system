## Overview

- The button component is used to action navigation to a web link or to an anchor tag on the same page or different page. 
- The buttons categories are primary, secondary, secondary-light and purple. 
- Each category also has its variant styles like normal, outline, text and expanded options to be used in different scenarios.

## Default example

### Primary

Primary buttons are used for primary actions on a webpage. For example, submit button on a form, or other critical actions in a user workflow.
It should only be used once on a webpage.

## How and when to use

### States

Default - the button text is not underlined. <br />
Hover - the button text is underlined. The background and text color may change depending on the variant.

### Breakpoints

Breakpoints have no effect on the button component and it visually remains the same.

## Accessibility

Accessibility colour contrast test for all buttons showed a WCAG AA Pass.
