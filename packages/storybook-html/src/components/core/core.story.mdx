# Core

This package provides [Sass](https://sass-lang.com/) variables, mixins, and
functions. 

> All other design system packages depend on this package for 
> consistency.

## Setup

To use `core` when developing your styles, you must use [`sass` (Dart Sass)](https://sass-lang.com/install)
to preprocess your stylesheets. See [getting started](/?path=/docs/introduction-getting-started--page).

### 1. Install the package

With NPM:  
```shell
npm i @uqds/core
```

With Yarn:  
```shell
yarn add @uqds/core
```

### 2. Use (import) the module in your project

In a SCSS/Sass file:

```scss
@use '@uqds/core/src/scss/_global' as core;

.example-class {
    ...
    font-family: core.$font-body;
    ...
}
```
