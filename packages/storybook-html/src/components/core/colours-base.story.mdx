# Base Colour palette

To maintain brand consistency you should reference the utility colours provided using Variable names over actual colour values shuch as Hex or RGB. 

- Base colours are used to create all variations throughout the design system.
- Do not create random colours, always start with a base colour reference.
- Utility colour guidelines should be followed before using these values
