## Written style guide

Reference the offical [Written style guide](https://marketing-communication.uq.edu.au/written-style-guide) for detailed instructions for writing content.  

### Capitalization / Letter-case

Use sentence case/minimal capitalisation in headings and subheadings. Use a capital for the first letter of the first word, with subsequent words in lower case.

Sentance case

All-caps text — meaning text with all the letters capitalised — is best used sparingly.

### Truncation

Line Length.
