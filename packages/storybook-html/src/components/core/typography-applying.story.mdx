# Applying Typography
Most common use cases have already been setup in the basic HTML elements and components. 

Follow existing examples or reference the chart below when configuring new websites and componets. 

* The font weights used are Regular (400) and Medium (500). 
* Italic style should be used when you need to emphasize certain words in a sentence or a statement.

### Roboto

* Roboto is the primary font used on websites and applications
* Roboto can be used for headings and body text. 

### Merriweather

* Merriweather can be used for pull out quotes and to break up long headings.
* Merriweather is ***not to be used for body text***.

### Montserrat

* Montserrat is used for main page (banner) heading.
* Montserrat is ***not to be used for body text***.
