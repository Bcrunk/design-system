# Side nav

<StatusBadge.Development />

## Overview

A vertical list of links for site navigation typically placed next to the body content. On smaller viewports, the side nav shifts underneath the page content. 

## Default example

### Single-level

<Canvas withSource="none">
  <Story id="components-side-nav--side-nav-single-level" />
</Canvas>

```html
<!-- Side nav single-level -->
<div class="uq-side-nav">

  <!-- Parent page title -->
  <h3 class="uq-side-nav__title">
    <a href="#" class="uq-side-nav__title-link">Parent page title</a>
  </h3>

  <!-- Side nav -->
  <nav class="uq-side-nav__container" aria-label="Side navigation">
    <ul class="uq-side-nav__list uq-side-nav__list--level-1">
      <li class="uq-side-nav__list-item uq-side-nav__list-item--active">
        <a href="#" class="uq-side-nav__link">Item 1</a>
      </li>
      <li class="uq-side-nav__list-item">
        <a href="#" class="uq-side-nav__link">Item 2</a>
      </li>
      <li class="uq-side-nav__list-item">
        <a href="#" class="uq-side-nav__link">Item 3</a>
      </li>
    </ul>
  </nav>
  
</div>
```

## When and how to use

- Use when there is related links to the webpage content. 
- Use a grid layout to place the side nav component next to the webpage content.

## Accessibility

Add `aria-label` tag to the side nav component `<nav>` element.
