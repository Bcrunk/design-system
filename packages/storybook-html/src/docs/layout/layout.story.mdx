# Layout

The *UQ Design System*'s responsive grid adapts to different screen sizes and
orientation.

## Breakpoints

| Name | Breakpoint range                   | Columns | Margins           | Gutters        |
|------|----------------------------------- | ------- | ----------------- | -------------- |
| *    | 0—25.9375rem (** ~0—415px **)      | 6       | 1.5rem (~24px)    | 1rem (~16px)   |
| XS   | 26—33.9375rem (** ~416px—  **)     | ''      | ''                | ''             |
| SM   | 34—0 rem (** ~544—767px **)        | ''      | ''                | ''             |
| MD   | 48—63.9375rem (** ~768—1023px **)  | 8       | 4rem (~64px)      | 1.5rem (~24px) |
| LG   | 64—89.9375rem (** ~1024—1439px **) | ''      | ''                | ''             |
| XL   | 90rem—∞ (** ~1440px—∞ **)          | 12      | 9.375rem (~150px) | 2rem (~32px)   |

*Pixel units (px) provided are based on a default browser text size of 16px.*  
\* Mobile screens should be designed for this base range. We recommend creating
320px width (artboard) mockups.
