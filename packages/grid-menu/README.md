# `@uqmchu/grid-menu`

> Grid menu component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqmchu/grid-menu
```

With NPM:
```shell
npm i @uqmchu/grid-menu
```

---

## Dependency graph

@uqmchu/grid-menu

├─ @uqmchu/foundation-global
