# `@uqds/grid-menu`

> Grid menu component

## Usage

### Installation

With Yarn:
```shell
yarn add @uqds/grid-menu
```

With NPM:
```shell
npm i @uqds/grid-menu
```

---

## Dependency graph

```shell
@uqds/grid-menu
├─ @uqds/core
├─ @uqds/foundation-global
│  └─ @uqds/core
```
